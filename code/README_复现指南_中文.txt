  三、核心算法详解

  3.1  复算子 K(Ω) —— 论文核心创新
  ------------------------------------------------------------------
    问题：15 维全耦合系统（x1 5 系数 + x2 5 系数 + q 5 系数）
         每步 Newton 迭代都需构造 15×15 Jacobian，计算量大。

    思路：电路方程为线性（RLC+NIC），可在频域代数消元：
          κₑ·q'' + σ·q' + κᴄ·q = θ·(x₁' - x₂')
          ├─ 频域 → Qₙ = θ·j·n·Ω / (-κₑ·(nΩ)² + j·σ·nΩ + κᴄ) · (X₁ₙ - X₂ₙ)
          └─ 代入力反馈 → 电磁力 = θ²·D·A⁻¹·D·(x₁ - x₂) ≡ K(Ω)·(x₁ - x₂)

    复算子定义：
          K(Ω) = θ²·Ω² / (κₑ·Ω² - j·σ·Ω - κᴄ)
               = K_r(Ω) + j·K_i(Ω)

    等效机械参数：
          C_eq(Ω) = K_i(Ω) / Ω      等效阻尼
          K_eq(Ω) = K_r(Ω)          等效刚度
          M_eq(Ω) = -Re{K(Ω)} / Ω²  等效惯容

    效果：系统从 15 维降至 10 维，Jacobian 从 15²=225 降至 10²=100，
          计算量降低约 55%，且完全等价（已通过 Validate_06 验证）。

  3.2  四级优化流水线（unified_optimization.m）
  ------------------------------------------------------------------
    Phase 1 — 算子层 LHS 快筛（秒级）
      · 在 K(Ω) 算子域内用 5000 个 LHS 样本评估
      · 无需运行 HBM，纯粹代数运算
      · 五目标加权评分函数：
          - 低频等效刚度 KL  （越小越好）
          - 低频等效阻尼 CL  （越大越好，目标 > 0.08）
          - 低频等效惯容 ML  （越大越好，目标 > 0.30）
          - 高频等效刚度 KH  （越大越好，目标 > 0.03）
          - 高频等效阻尼 CH  （越小越好）
          - 锐度惩罚（Ceq 峰谷比 > 12）
      · 选出 Top-30 进入 Phase 2

    Phase 2 — 系统层 FRF 验证（分钟级）
      · 对 Top-30 候选运行完整 15 维 HBM + Newton（180 频点）
      · 计算真实力传递率
      · 弧长延拓提供初值接力（保证收敛）
      · 抽查 Floquet 稳定性
      · 精选 Top-8 进入 Phase 3

    Phase 3 — fminsearch 局部精修（分钟级）
      · Sigmoid 变换实现有界搜索（约束参数在 [0.02, 3.0]³）
      · 目标函数：J = TF_peak + 稳定性惩罚项
      · 每个候选最多 80 次迭代
      · 保留 J-最优和 TF-最优两组解

    Phase 4 — 多候选精细验证 + 出版级对比图
      · 350 频点 + 完整 Floquet 稳定性扫描
      · 计算纯机械基准（λ=0，无电路）
      · 6 面板对比图 + 候选对比图

  3.3  谐波收敛性验证（Verify_Harmonic_Convergence.m）
  ------------------------------------------------------------------
    验证 3 谐波（0/1/3）截断的充分性：
      测试点 1：fold 分岔点附近（Ω ≈ 0.8）
      测试点 2：力传递率峰值处
      测试点 3：高激励幅值（Fw = 0.05 时）

    比较 3 谐波 HBM 与 5 谐波 HBM 的解。
    预期：|A₅| / |A₁| < 1%，第 5 谐波比基频低 2 个数量级以上。
    原因：QZS 对称几何使恢复力为奇函数 → 仅有奇次谐波。

  3.4  分岔类型分类（Run_Bifurcation_Classification.m）
  ------------------------------------------------------------------
    分析 Floquet 乘子在复平面上的穿越模式：
      · 实乘子从 (+1, 0) 穿出 → Fold（鞍结）分岔 → 跳跃现象
      · 实乘子从 (-1, 0) 穿出 → Flip（倍周期）分岔 → 次谐波振荡
      · 共轭复乘子穿出单位圆 → Neimark-Sacker 分岔 → 准周期调制

    在 Ω = 0.5, 1.0, 2.0 三个频率下扫力，产出：
      · 带分岔标注的幅频图
      · 主导乘子轨迹 Argand 图

  3.5  有源功率分析（compute_nic_power.m）
  ------------------------------------------------------------------
    计算 NIC 注入的周期平均有源功率：
      P_active = <q̇ · u_nic>_T  （无量纲）
    转换为有量纲瓦特（W），验证在运放线性范围内（典型 OPA 系列：
    ±10 mA / ±10 V，对应约 10-100 mW 线性输出）。

  四、物理模型与参数约定

  4.1  系统构型
  ------------------------------------------------------------------
    · 上层质量 m₁：QZS 机构 + 水平弹簧
    · 下层质量 m₂：QZS 机构 + 电磁耦合（VCM）
    · VCM 动圈并联于 RLC+NIC 电路
    · NIC（负阻抗变换器）：合成负电阻以抵消线圈电阻

  4.2  无量纲化动力学方程
  ------------------------------------------------------------------
    机械方程（2 自由度）：
      x₁'' + μ(x₁'' - x₂'') + F_QZS₁(x₁, x₁') + F_em = Fw·cos(Ω·τ)
      x₂'' -  (x₁'' - x₂'') + F_QZS₂(x₂, x₂') - F_em = 0

    电路方程（被 K(Ω) 消元）：
      κₑ·q'' + σ·q' + κᴄ·q = θ·(x₁' - x₂')

  4.3  参数向量 sysP = [11 元素]
  ------------------------------------------------------------------
    序号  符号      含义                   默认值/范围
    ────  ────────  ────────────────────  ──────────────────
     1    be1       上层线性刚度            1.0（固定）
     2    be2       下层线性刚度            α₂（由 β 导出）
     3    mu        质量比 m₂/m₁            0.2
     4    al1       上层刚度偏置            α₁ - be1 = -0.95
     5    ga1       上层立方非线性系数      1.5
     6    ze1       下层阻尼比              0.05
     7    lam       机电耦合 θ²             0.18（固定）
     8    kap_e     无量纲电感              优化值：1.5222
     9    kap_c     无量纲逆电容            优化值：0.5743
    10    sigma     无量纲总电阻            优化值：1.1506
    11    ga2       下层立方非线性系数      1.5

  4.4  关键无量纲参数定义
  ------------------------------------------------------------------
    符号      定义              数值      含义
    ────────  ────────────────  ────────  ──────────────────
    μ         m₂/m₁             0.2       质量比
    β         k₂ᵥ/k₁ᵥ           2.0       下层/上层线刚度比
    K₁        k₁h/k₁ᵥ           1.0       上层水平/垂直弹簧比
    K₂        k₂h/k₁ᵥ           0.2       下层水平/垂直弹簧比
    U         L₀/δₛ             2.0       几何尺度
    Lg        L₂/L₀             4/9       杆长比
    v         α₀                2.5       预压缩参数

    导出量：
    α₁ = v - 2·K₁·(1-Lg)/Lg         上层线性刚度系数
    α₂ = β - 2·K₂·(1-Lg)/Lg         下层线性刚度系数
    γ₁ = K₁/(U²·Lg³)                上层立方非线性系数
    γ₂ = K₂/(U²·Lg³)                下层立方非线性系数

  4.5  HBM 谐波截断
  ------------------------------------------------------------------
    每自由度用 5 个 HBM 系数表示：[a₀; a₁; b₁; a₃; b₃]
    对应谐波：0 次（直流）、1 次（基频）、3 次（三倍频）。
    偶次谐波恒为零（QZS 恢复力为奇函数 → Duffing 立方非线性）。

    全耦合模型：   3 DOF × 5 系数 = 15 未知量 + Ω/Fw = 16 维
    复算子消元模型：2 DOF × 5 系数 = 10 未知量 + Ω/Fw = 11 维

  七、常见问题

  Q: 运行报 "Undefined function or variable"
  A: 请先执行 init_path()。每次重新打开 MATLAB 都需要执行一次。

  Q: Newton 求解器在某些频点不收敛
  A: 这是正常现象，FRF 的转折点附近 Newton 可能失败。弧长延拓算法
     (branch_follow2.m) 专门处理这种情况。请使用 run 脚本中的封装
     而非直接逐点调用 Newton。预期收敛率 > 95%。

  Q: 得到的降幅不是 62.6%
  A: 请确认 Fw=0.008, 频点数 ≥ 350, 频段 [0.2, 6.0] 对数均匀分布。
     较粗的网格可能漏掉真正的 TF 峰值。可用 verify_peak_reduction.m
     独立验证（该脚本使用了精确的最优参数）。

  Q: 内存不足
  A: 关闭其他 MATLAB 图窗。Phase 1 的 5000 样本 LHS 矩阵运算
     内存最大（约 1-2 GB）。若仍有问题可减少 Nsamp_phase1 至 2000。

  Q: 生成的图与论文不完全一致
  A: 请确认随机种子已设置（unified_optimization.m 中 rng(42)），
     LHS 采样在固定种子下完全可重复。
